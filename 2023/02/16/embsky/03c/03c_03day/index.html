一.温故知新

    运算符
        算术运算符		+ - * / %
            注意 : 除法运算符中除数不能为0
        关系运算符		> < == >= <= !=
            注意 : 
            1.一个表达式经过关系运算符的运算最后是真假值
            2.判断两个值是否相等用==
            3.if(a == 13) 建议 if(13 == a)
        逻辑运算符		&& || !
            注意 :
            1.一个表达式经过逻辑运算符的运算最后是真假值
            2.条件1 && 条件2
                条件1为真,会执行条件2
                条件1为假,不会执行条件2
            3.条件1 || 条件2
                条件1为真,不会执行条件2
                条件1为假,会执行条件2
        赋值运算符		= += -= *= /= &= |= ...
            注意 : 先计算右边的项,再和左边结合
        位运算符		<< >> & | ~ ^
            注意 : 操作数符号的问题
        条件运算符		? :
            注意 : 是C语言中唯一的一个三目运算符
        下标运算符		[]
        指针运算符		* &
        分量运算符		. ->
        求字节大小运算符sizeof()
            注意 : sizeof不是函数,是一个运算符
        强制类型转换	()
            注意 : 会在指针的时候使用
        自增自减		++ --
        逗号运算符		,

逗号运算符

    如果一个表达式使用了逗号运算符
        如果没有加小括号,那么会把第一个项的结果赋值给变量
        如果有加小括号,那么会把最后一个项的结果赋值给变量

    if
1.if else是C语言的关键字
2.if使用方法
    语句1
    if(条件)
    {
        语句2
    }
    语句3

语句1->条件的判断(真)->语句2->语句3
        |
        (假)->语句3
3.if else使用方法
    语句1
    if(条件)
    {
        语句2
    }
    else
    {
        语句3
    }
    语句4
语句1->条件的判断(真)->语句2->语句4
        |
        (假)->语句3->语句4
4.if else if else if使用方法
    语句1
    if(条件1)
    {
        语句2
    }
    else if(条件2)
    {
        语句3
    }
    else if(条件3)
    {
        语句4
    }
    else
    {
        语句5
    }
    语句6
语句1->条件1的判断(真)->语句2->语句6
        |
        (假)->条件2的判断(真)->语句3->语句6
                |
                (假)->条件3的判断(真)->语句4->语句6
                        |
                        (假)->语句5->语句6
5.使用if需要注意的问题
    1>if关键字之后必须要写小括号,小括号内必须写内容
    2>else关键字之后不能直接加小括号写条件,需要写成else if
    3>if(条件)之后可以直接加 ; 
        if(条件);
        {
            语句
        }
        等价于
        if(条件)
        {
            ;
        }
        {
            语句
        }
    4>if(条件)之下可以不加大括号
        if(条件)
            语句1
            语句2
        等价于
        if(条件)
        {
            语句1
    	}
